# 🔍 Fraud Detection API

A production-grade ML-powered REST API that detects fraudulent credit card transactions in real time.

## 🚀 Live API
Run locally at http://127.0.0.1:8000/docs

## 📊 Model Performance
- Accuracy: 100%
- Fraud Precision: 97%
- Trained on 284,807 real transactions

## 🛠️ Built With
- Python
- FastAPI
- scikit-learn
- RandomForest Classifier
- Uvicorn
